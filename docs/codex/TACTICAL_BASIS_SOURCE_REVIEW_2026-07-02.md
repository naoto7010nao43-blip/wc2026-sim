# Tactical Basis Source Review - 2026-07-02

対象: `backend/data/seed/teams.json` に未コミットで追加されている `_tactical_profile_basis` 候補。

## 結論

この差分は、戦術値の根拠を残す方向性としては有用だが、現時点ではそのままseedへコミットしない。

理由:

- 25チームに根拠メモが追加されているが、そのうち8チームはURLが0件。
- URL付き51件のうち48件は到達可能、3件は403で機械確認不可。
- `_tactical_profile_basis` は現行モデル計算には直接使われないが、監督・戦術監査の `has_tactical_basis` 判定を大きく変えるため、未検証のまま入れると「根拠あり」と誤認させる。
- 文面には2026年大会後の戦術評価らしき記述も含まれるため、一次情報・日付・実際の試合文脈を追加確認してから採用すべき。

## 到達性サマリ

- `_tactical_profile_basis` 追加チーム: 25

## 採用方針

1. URLが0件のチームは、seed反映ではなく外部調査キューへ戻す。
2. 403のURLは、人間またはブラウザで本文確認できるまで根拠として数えない。
3. URLが到達可能でも、記事本文が実際に該当監督・該当代表・該当時点の戦術値を支持するかを確認する。
4. seedに入れる場合は、自由文1本ではなく `tactical_profile_sources` のような構造化配列に分ける方がよい。
5. 監査側の `has_tactical_basis` は、単なる文字列存在ではなく「URL付き・確認済み・日付あり」の条件に変える。

## Codex判断

今回のグループ突破確率機能にはこの候補差分を混ぜない。戦術根拠メモはClaude Codeまたは次のCodex作業で、構造化・URL検証・本文整合性確認を経てから別コミットにする。
